# CAN Domain Knowledge

## Overview
The CAN domain covers vehicle network telemetry and bus signals transmitted over the Controller Area Network. This knowledge base is used by the analyzer to interpret CAN parameter behavior and detect anomalies.

## Key CAN Signals
- `CAN_RPM`: Engine speed
- `CAN_SPEED`: Vehicle speed
- `CAN_ENG_TEMP`: Engine temperature
- `CAN_ACCEL`: Accelerator position

## Use Cases
- Detect overheating or engine thermal issues
- Validate speed and RPM consistency
- Monitor driveability and sensor faults
