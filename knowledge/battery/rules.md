# Battery Rules

## Purpose
This file stores battery-specific anomaly detection rules. It supports identifying critical battery conditions and operational limits.

## Rule Definitions
- `BATT_KN_001`: Low battery voltage below safe operating threshold.
- `BATT_KN_002`: Over-temperature event when battery temperature exceeds safety range.

## Guidance
Use battery rules to surface issues with power delivery, charging behavior, and thermal distress.
